# Prediksi Harga Rumah

Proyek ini bertujuan untuk memprediksi harga rumah menggunakan teknik Machine Learning.

## Prasyarat

Pastikan Anda sudah menginstal Python di sistem Anda. Proyek ini direkomendasikan untuk dijalankan di dalam lingkungan virtual (*virtual environment*).

## Cara Penggunaan

Ikuti langkah-langkah di bawah ini untuk menyiapkan lingkungan pengembangan dan menjalankan proyek:

### 1. Clone Repositori
Pertama, clone repositori ini ke mesin lokal Anda:
```bash
git clone https://github.com/Wildan-bin/Prediksi-Harga-Rumah.git
cd Prediksi-Harga-Rumah
```

### 2. Buat Virtual Environment (venv)
Gunakan modul `venv` bawaan Python untuk membuat lingkungan terisolasi:
```bash
# Windows
python -m venv venv

# macOS/Linux
python3 -m venv venv
```

### 3. Aktifkan Virtual Environment
Setelah dibuat, aktifkan lingkungan tersebut:
```bash
# Windows (Command Prompt)
venv\Scripts\activate

# Windows (PowerShell)
.\venv\Scripts\Activate.ps1

# macOS/Linux
source venv/bin/activate
```

### 4. Instal Dependensi
Gunakan `pip` untuk menginstal semua pustaka yang diperlukan dari file `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 5. Jalankan Proyek
Setelah semua dependensi terinstal, Anda dapat menjalankan notebook atau skrip Python yang tersedia:
```bash
# Jika menggunakan Jupyter Notebook
jupyter notebook
```

## Struktur Proyek
- `requirements.txt`: Daftar pustaka yang diperlukan.
- `.ipynb` atau `.py`: File utama untuk pemrosesan data dan pemodelan.
- `data/`: (Opsional) Folder untuk menyimpan dataset.
